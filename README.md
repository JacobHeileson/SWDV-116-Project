## Project Overview

The website allows users to explore outdoor experiences such as hunting, fishing, and camping.
It also includes an interactive estimate calculator where visitors can select species, camping options, and other details to generate a cost estimate.
This project was developed as part of the SWDV 116 curriculum to practice building multi-page websites, organizing project files, and implementing basic HTML, CSS, and JavaScript.
